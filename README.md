#### This dataset is a Heart Indicator dataset from Kaggle (https://www.kaggle.com/datasets/kamilpytlak/personal-key-indicators-of-heart-disease/discussion/448736/data).
#### In this dataset, it is a survey based Q/A of the many adult respondents about their health (about 246,022 respondents) with 40 rows.
#### This project aims to predict an indication of heart disease or not, with many factors like BMI, PhysicalHealth, SmokerStatus, Having Agina, and much more.
